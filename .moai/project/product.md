# BookSummary 프로덕트 문서

## 프로젝트명 및 슬로건

**BookSummary** - 책 요약 및 리뷰 영상 자동 생성 파이프라인

자동화된 YouTube 콘텐츠 생성으로 책 홍보의 시간과 비용을 줄입니다.

## 프로젝트 설명

BookSummary는 책의 내용을 자동으로 분석하고 두 가지 스타일의 YouTube 영상으로 변환하는 완전 자동화 파이프라인입니다. Claude AI를 이용한 지능형 요약, 멀티 제공자 TTS(Text-to-Speech), 이미지 자동 수집, 그리고 NotebookLM을 활용한 리뷰 영상 생성으로 책 마케팅을 효율화합니다.

**존재하는 이유:**
- 책 홍보 영상 제작에 소요되는 시간 단축 (수동 작업 10시간 → 자동화 20분)
- 다국어 콘텐츠 동시 생성 (한글/영문 자동 제작)
- 일관된 영상 품질과 포맷 유지
- 개인 크리에이터의 생산성 극대화

## 핵심 기능

### 1. 자동 요약 생성
- Claude 3.5 Sonnet API를 이용한 5분 분량 요약 텍스트 생성
- 책 제목, 저자, 주요 내용 자동 분석
- Hook, Body, Bridge 구조로 시청자 집중력 최대화

### 2. 멀티 제공자 TTS (Text-to-Speech)
- OpenAI TTS: 빠르고 자연스러운 음성 (nova, alloy, echo, fable 등)
- Google Cloud Neural2: 한국어 발음 최적화
- Replicate xtts-v2: 오픈소스 기반 다국어 지원
- 사용자 선택 음성으로 최적 품질 보장

### 3. 이미지 자동 수집
- Pexels, Pixabay, Unsplash에서 책 주제 관련 이미지 100개 자동 다운로드
- 책 커버 이미지 자동 추출
- 무드 이미지로 시각적 다양성 확보

### 4. 두 가지 영상 스타일 제작

#### 4-1. Summary+Video 스타일
- 구조: 5분 요약 영상 + NotebookLM 리뷰 영상
- 길이: 약 10-15분 단일 영상
- 용도: 책 핵심 내용 설명 및 감상 공유

#### 4-2. 일당백 스타일
- 구조: Part 1 (저자/배경) + Part 2 (줄거리) 연결
- 형식: 에피소드 형태의 멀티파트 영상
- 용도: 깊이 있는 책 내용 설명

### 5. YouTube 자동 업로드
- 메타데이터 자동 생성 (제목, 설명, 태그)
- 썸네일 자동 생성 (PNG → JPG 변환)
- 비공개/공개 설정 자동화
- 재생목록 자동 추가

### 6. 제휴 링크 관리
- Amazon Associates 링크 자동 추가 (태그: joohans-20)
- 알라딘 TTB 제휴 링크 통합 (ID: ttbjsong12301317001)
- 일괄 댓글 추가 (고정 댓글)
- 링크 유효성 자동 검증

### 7. 다국어 지원
- 한글/영문 동시 생성
- 자동 번역 매핑 (200+ 책/저자)
- 언어별 최적화된 음성 선택

## 사용 사례 및 워크플로우

### 워크플로우 1: Summary+Video 영상 제작
```
1. 책 정보 입력 (제목, 저자)
2. 이미지 100개 자동 수집
3. Claude API로 5분 요약 생성
4. TTS 제공자 선택 (OpenAI/Google/Replicate)
5. 요약 오디오 생성
6. NotebookLM 리뷰 오디오/비디오 준비
7. 이미지 + 오디오로 최종 영상 합성
8. 썸네일 생성
9. YouTube 메타데이터 작성
10. 업로드 (비공개 → 검토 → 공개)
11. 제휴 링크 댓글 추가
```

### 워크플로우 2: 일당백 스타일 영상 제작
```
1. YouTube 자막 다운로드 (Part 1, Part 2)
2. Part 1, Part 2 비디오 + 인포그래픽 준비
3. 비디오 순차 연결 (Part 1 video → Part 1 info → Part 2 video → Part 2 info)
4. 최종 에피소드 영상 생성
5. 썸네일 생성
6. YouTube 메타데이터 작성 (고정 포맷)
7. 업로드
8. 제휴 링크 댓글 추가
```

### 워크플로우 3: 배치 처리 (여러 책)
```
1. CSV에서 책 목록 읽기
2. 각 책별 이미지 수집, 요약, 영상 제작 (병렬)
3. 업로드 전 수동 검수 (썸네일, 제목)
4. 일괄 업로드
5. CSV 상태 업데이트
```

## 현재 상태

### 생산 통계
- **총 업로드 영상:** 246개 이상
- **자동화 수준:** 95% (사용자 개입 최소화)
- **운영 기간:** 지속적 운영 중

### 주요 성과
- **시간 절약:** 영상당 2-3시간 → 20분으로 단축
- **다국어 지원:** 한글/영문 동시 생성으로 구독자 확대
- **안정성:** 자동 오류 처리 및 재시도 메커니즘
- **유지보수성:** 스크립트 모듈화로 유연한 업데이트 가능

### 현재 한계
- YouTube API 일일 할당량: 10,000 units/day (약 50-60개 영상 업로드)
- NotebookLM 의존성: 음성/비디오 수동 준비 필요
- 썸네일: AI 생성이 아닌 사용자 제공 이미지 기반
- 자막: 한글↔영문 자동 번역 미지원 (별도 처리)
