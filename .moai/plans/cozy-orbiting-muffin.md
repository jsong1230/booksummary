# YouTube 구독자 성장 전략 (100~500 -> 1,000+)

## Context

Beyond Page 채널은 약 3개월간 70개 영상을 업로드하고 100~500명의 구독자를 보유하고 있습니다. 자동화 파이프라인이 강력하나, 콘텐츠 전략/SEO/커뮤니티 활용이 부족합니다. 459개 도서 중 387개가 미처리되어 대규모 백로그가 경쟁 우위입니다.

---

## Phase 1: 즉시 실행 (이번 주)

### 1-1. 태그 스터핑 수정 (SEO 즉시 개선)
- **파일**: `src/08_create_and_preview_videos.py` (lines 974-1039)
- **문제**: 모든 영상에 Harvard, NYT, CNN, 서울대, 고려대, 연세대 등 무관한 태그 일괄 적용 -> YouTube 알고리즘 패널티
- **수정**: 조건부 적용으로 변경
  - Tier 1 (필수 5개): 책 제목(한/영) + 저자(한/영) + "책리뷰"
  - Tier 2 (장르 3-4개): 기존 장르 감지 활용하되 해당 장르만
  - Tier 3 (도서 특화 3-4개): 실제 수상작만 수상 태그, 실제 권장도서만 대학 태그
  - Tier 4 (채널 2개): "BeyondPage" + 콘텐츠 유형
  - 최대 15개 태그 제한 (현재 30개+)

### 1-2. Shorts 일괄 생성 (조회수 급증 기대)
- 기존 업로드 영상 중 조회수 상위 10개에 대해 Shorts 생성
- `python src/26_generate_shorts.py` 실행 -> 영상당 3개 = 30개 Shorts
- Shorts 제목 개선: "이 책이 인생을 바꿨다" (고정) -> 책별 맞춤 훅 카피

### 1-3. 분석 기준선 확보
- `python src/22_analytics_recommendations.py --days 90` 실행
- `python src/18_analyze_optimal_upload_time.py` 실행
- 어떤 장르/도서가 잘 되는지 데이터 기반 판단

### 1-4. 기존 댓글 전수 응답
- 70개 영상의 미응답 댓글에 모두 답글
- 하트 찍기 (비용 0, 효과 큼)

## Phase 2: 이번 달 (4주 내)

### 2-1. 제목 공식 개선
- **파일**: `src/08_create_and_preview_videos.py`, `src/utils/title_generator.py`
- **현재**: `[핵심 요약] 침팬지 폴리틱스: 프란스 드 발 (5분 핵심 요약·AI 심층 분석·핵심 주제·인사이트·정리)`
- **개선안 (한글)**: `프란스 드 발이 밝힌 권력의 본질 | 침팬지 폴리틱스 핵심 요약`
- **개선안 (영문)**: `Frans de Waal on the Nature of Power | Chimpanzee Politics Summary`
- **효과**: 훅 카피 전면 배치 -> CTR 향상, 검색어(책 제목) 유지

### 2-2. Description 템플릿 개선
- **파일**: `src/08_create_and_preview_videos.py` (lines 333-497)
- **현재 문제**: 첫 2-3줄이 "NotebookLM과 AI를 활용하여..." 보일러플레이트 -> 검색 결과에서 낭비
- **개선**: 첫 줄에 책 특화 훅 문장 배치, AI 언급 제거

### 2-3. 업로드 빈도/스케줄 최적화
- **현재**: 주 5-6개 (무작위 시간)
- **개선**: 주 4개 장편 + 매일 1 Shorts
  - 월: Summary+Video 한글 (도서 A)
  - 화: Summary+Video 영문 (도서 A)
  - 목: Summary+Video 한글 (도서 B)
  - 금: Summary+Video 영문 (도서 B)
  - 매일: Shorts 1개 (기존 영상에서 추출)
- 영문 영상 주 4->2개로 축소 (한국어에 집중)

### 2-4. 작가 스포트라이트 시리즈 시작
- 첫 시리즈: 헤르만 헤세 (데미안 + 싯다르타 이미 업로드 -> 유리알 유희 추가)
- 같은 작가 영상을 연속 업로드해 몰아보기 유도
- 플레이리스트 자동 관리: `src/27_manage_playlists.py` 활용

### 2-5. 장르 다변화 테스트
- CSV의 95%가 고전문학 -> 주 1회 비고전 도서 테스트
- 심리학/자기계발 (한국 YouTube 수요 높음)
- CTR/조회수 비교 후 비율 조정

## Phase 3: 3개월 전략

### 3-1. 콘텐츠 퍼널 구축
```
Shorts (15-30초)     -> 신규 시청자 유입
  ↓
Summary+Video (5-7분)  -> 구독 전환
  ↓
일당백 (13-16분)       -> 충성 시청자 구축
```
- 일당백은 Summary+Video에서 조회수 높은 도서만 선택 (데이터 기반)
- Shorts에서 "전체 리뷰는 채널에서" CTA 텍스트 오버레이 추가

### 3-2. 커뮤니티 활성화 (500명 달성 후)
- `src/28_community_posts.py` 활성화
  - 월: 명언 포스트 (최근 업로드 도서)
  - 수: 투표 (다음 리뷰할 도서 4지선다)
  - 금: 채널 업데이트/비하인드
- 댓글에서 "다음에 리뷰해줄 도서" 요청 추적 -> CSV에 별도 컬럼

### 3-3. 387개 백로그 활용한 "라이브러리 효과"
- 3개월 후 목표: 150개+ 영상 (검색 가능한 고전문학 도서관)
- 하나의 도서로 검색 유입 -> 유사 도서 발견 -> 구독
- 알고리즘이 추천할 "관련 영상" 풀 확대

### 3-4. 썸네일 A/B 테스트 프로세스
- 도서별 2개 썸네일 변형 생성 (질문형 vs 진술형)
- 변형 A로 업로드 -> 48시간 후 CTR 확인
- CTR 3% 미만이면 변형 B로 교체
- 결과를 CSV에 기록하여 패턴 학습

### 3-5. 채널 분리 판단 (500명+ 도달 시)
- 영문 콘텐츠가 독자적 성장하면 -> 영문 전용 채널 분리
- 성장 안 되면 -> 주 1회로 축소, SEO 보조 콘텐츠로 활용

## 검증 방법

1. **1주 후**: 태그 수정 후 신규 영상 노출수(Impressions) 변화 확인
2. **2주 후**: Shorts 업로드 후 채널 조회수/구독자 증가 추세 확인
3. **4주 후**: 제목 공식 변경 영상 vs 기존 공식 영상 CTR 비교
4. **3개월 후**: 구독자 1,000명 달성 여부 + 영상별 평균 조회수 추세

## 핵심 원칙

- **데이터 기반 의사결정**: 감이 아닌 분석 스크립트 결과로 판단
- **양보다 질**: 주 6개 -> 4개, 대신 제목/썸네일/SEO 최적화
- **퍼널 사고**: Shorts -> Summary -> 일당백 단계별 깊이
- **백로그 = 경쟁 우위**: 387개 미처리 도서가 장기 성장 자산
