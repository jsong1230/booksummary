# English Episode (1DANG100) Thumbnail Prompt Template

## How to Use

Replace the placeholders below and use with an AI image generator (DALL-E, Midjourney, etc.).

| Placeholder | Description | Example |
|------------|-------------|---------|
| `{author_name}` | Author's name | Haruki Murakami |
| `{book_title}` | Book title | Norwegian Wood |
| `{hook_sentence}` | Core message as a curiosity-inducing hook | Why does youth only make sense through loss? |
| `{character_description}` | Description of author or main character | A young man leaning by a window, listening to a record |
| `{world_background}` | Book's world/setting for background | Rainy 1960s Tokyo streets |

---

## Prompt (copy and use)

```
{author_name}-{book_title} 1DANG100 style English thumbnail.

Format: 16:9, 1920x1080
Background: White with faint grid texture
Frame: Blue geometric infographic-style border (circles + rectangles on edges, #1A73E8)
Top text: "{author_name}: {hook_sentence}" — Bold black, very large
Bottom illustration: Watercolor + pen-and-ink style of {character_description}. Background: {world_background}.
Overall feel: Clean, bright, illustration as focal point. No photos.
```

---

## Checklist

Verify all 6 items before uploading:

- [ ] Is the text a "hook copy" rather than the book title?
- [ ] Is it illustration style? (No photos)
- [ ] White background + blue geometric frame?
- [ ] Text large and bold at the top?
- [ ] Does it include a person or animal?
- [ ] Is it visually simple? (3 elements or fewer)

---

**Bad**: `Norwegian Wood — Haruki Murakami`

**Good**: `Haruki Murakami: Why does youth only make sense through loss?`
