# 한글 일당백 썸네일 프롬프트 템플릿

## 사용 방법

아래 프롬프트에서 플레이스홀더를 교체하여 이미지 생성 AI(DALL-E, Midjourney 등)에 사용하세요.

| 플레이스홀더 | 설명 | 예시 |
|------------|------|------|
| `{author_name}` | 저자 이름 | 무라카미 하루키 |
| `{book_title}` | 책 제목 | 노르웨이의 숲 |
| `{hook_sentence}` | 핵심 메시지를 궁금증 유발 문장으로 | 청춘은 왜 상실로 완성되는가? |
| `{character_description}` | 저자 또는 책 캐릭터 묘사 | 창가에 기대어 LP판을 듣는 청년 |
| `{world_background}` | 책의 세계관 반영 배경 | 비가 내리는 1960년대 도쿄 |
| `{mood_color}` | 책 분위기에 맞는 배경 색감 | 진한 남색~검정 그라데이션 |
| `{light_source}` | 드라마틱 조명 효과 | 촛불의 따뜻한 빛이 중앙에서 퍼지는 느낌 |

---

## 프롬프트 (복사하여 사용)

```
{author_name}-{book_title} 일당백 스타일 한글 썸네일 만들어줘.

형식:
- 16:9, 1920x1080
- 배경: {mood_color}, {light_source}
- 파란색 인포그래픽 테두리 프레임 (#1A73E8)
- 상단 텍스트: "{author_name}: {hook_sentence}" — 굵은 화이트 한글, 크게, 그림자 효과로 가독성 확보
- 하단 일러스트: {character_description}의 수채화+펜화 스타일 일러스트. 배경: {world_background}.
- 전체 느낌: 어두운 배경에 조명이 강조. 드라마틱하고 시선을 끄는 느낌. 일러스트가 포인트. 사진 X.
```

---

## 체크리스트

업로드 전 아래 6가지를 모두 확인하세요:

- [ ] 텍스트가 책 제목이 아닌 "훅 카피"인가?
- [ ] 일러스트 스타일인가? (사진 X)
- [ ] 어두운 배경 + 드라마틱 조명인가?
- [ ] 텍스트가 상단에 크고 굵게 (화이트) 있는가?
- [ ] 인물 or 동물이 포함되어 있는가?
- [ ] 전체적으로 복잡하지 않은가? (요소 3개 이하)

## 배경 색감 가이드

| 책 분위기 | 추천 배경 | 추천 조명 |
|----------|----------|----------|
| 철학/사색 | 진한 남색~검정 그라데이션 | 촛불 또는 달빛 |
| 전쟁/비극 | 짙은 회색~적갈색 그라데이션 | 불꽃 또는 석양빛 |
| 사랑/감성 | 깊은 보라~남색 그라데이션 | 따뜻한 황금빛 |
| 모험/역사 | 짙은 갈색~카키 그라데이션 | 횃불 또는 새벽빛 |
| 공포/미스터리 | 검정~짙은 녹색 그라데이션 | 희미한 단일 광원 |
| 유머/풍자 | 진한 티얼~네이비 그라데이션 | 무대 스포트라이트 |

## 예시

**좋은 예**: `무라카미 하루키: 청춘은 왜 상실로 완성되는가?`
