# English Summary+Video Thumbnail Prompt Template

## How to Use

Replace the placeholders below and use with an AI image generator (DALL-E, Midjourney, etc.).

| Placeholder | Description | Example |
|------------|-------------|---------|
| `{author_name}` | Author's name | Viktor Frankl |
| `{hook_sentence}` | Core message as a curiosity-inducing hook (NOT the book title) | Can suffering still hold meaning? |
| `{illustration_subject}` | Description of author or main character | A man gazing at stars through barbed wire |
| `{thematic_background}` | Key symbol or setting from the book | Barbed wire fence, a single ray of light |
| `{mood_color}` | Background color matching book mood | Dark gray-to-crimson gradient |
| `{light_source}` | Dramatic lighting effect | A single ray of light descending from above |

---

## Prompt (copy and use)

```
Create a YouTube thumbnail in 16:9 ratio (1920x1080px).

LAYOUT:
- Background: {mood_color}, {light_source}
- Blue geometric frame elements on left and right edges (circles, rectangles - infographic UI style, #1A73E8)
- Top 35%: Large bold English text, white with drop shadow, centered
- Bottom 60%: Illustrated scene

TEXT (top area):
"{author_name}: {hook_sentence}"
Font: Bold sans-serif, very large, white with subtle drop shadow, high contrast

ILLUSTRATED SCENE (bottom area):
Watercolor and pen-and-ink illustration style.
Show: {illustration_subject} + {thematic_background}
Style: Soft watercolor with fine pen details. Dramatic lighting from the background. NOT photorealistic.

COLOR: Dark atmospheric background. Watercolor natural tones highlighted by dramatic lighting. Blue geometric frame (#1A73E8).

DO NOT include: photorealistic elements, plain white background, excessive text, channel branding.
```

---

## Checklist

Verify all 6 items before uploading:

- [ ] Is the text a "hook copy" rather than the book title?
- [ ] Is it illustration style? (No photos)
- [ ] Dark background + dramatic lighting?
- [ ] Text large and bold at the top (white)?
- [ ] Does it include a person or animal?
- [ ] Is it visually simple? (3 elements or fewer)

## Background Color Guide

| Book Mood | Recommended Background | Recommended Lighting |
|-----------|----------------------|---------------------|
| Philosophy/Reflection | Deep navy-to-black gradient | Candlelight or moonlight |
| War/Tragedy | Dark gray-to-crimson gradient | Flames or sunset glow |
| Love/Emotion | Deep purple-to-navy gradient | Warm golden light |
| Adventure/History | Dark brown-to-khaki gradient | Torchlight or dawn light |
| Horror/Mystery | Black-to-dark-green gradient | Dim single light source |
| Humor/Satire | Deep teal-to-navy gradient | Stage spotlight |
| Self-help | Deep navy-to-indigo gradient | Dawn light or spotlight |
