# Gems 썸네일 프롬프트 필수 조건

Beyond Page 채널 썸네일 생성 시 Gems 지시문에 반드시 포함해야 하는 3가지 조건입니다.

---

## 필수 조건 1: 훅 카피 (책 제목 그대로 X)

텍스트를 책 제목 그대로 넣지 마세요.
책의 **핵심 메시지를 궁금증 유발 문장**으로 변환하여 넣어야 합니다.

| 금지 (BAD) | 허용 (GOOD) |
|-----------|------------|
| `나는 고양이로소이다 — 나쓰메 소세키` | `나쓰메 소세키: 근대를 비판한 고양이` |
| `Man's Search for Meaning — Viktor Frankl` | `Viktor Frankl: Can suffering still hold meaning?` |
| `노르웨이의 숲 — 무라카미 하루키` | `무라카미 하루키: 청춘은 왜 상실로 완성되는가?` |

**Gems 지시문 추가 문구:**
```
썸네일 상단 텍스트는 책 제목을 그대로 사용하지 말 것.
반드시 책의 핵심 메시지를 독자의 호기심을 자극하는 한 문장으로 변환하여 작성할 것.
형식: "{저자명}: {궁금증 유발 문장}"
```

---

## 필수 조건 2: 수채화+펜화 스타일 (사진 리얼리즘 X)

일러스트 스타일을 반드시 **수채화+펜화(watercolor and pen-and-ink)**로 지정해야 합니다.
사진 리얼리즘이나 3D 렌더링은 채널 아이덴티티와 맞지 않습니다.

**Gems 지시문 추가 문구:**
```
일러스트 스타일: 반드시 수채화+펜화(watercolor and pen-and-ink) 스타일로 생성할 것.
사진 리얼리즘(photo-realistic), 3D 렌더링, 다크/무드 배경은 절대 사용 금지.
부드러운 수채화 터치 + 펜 세선 스케치 조합. 따뜻하고 약간 빈티지한 톤.
```

---

## 필수 조건 3: 파란색 기하학 인포그래픽 프레임

채널의 시각적 아이덴티티를 유지하기 위해 **파란색 기하학 인포그래픽 테두리 프레임**을 반드시 포함해야 합니다.

- 색상: `#1A73E8` (Google Blue) 또는 유사한 파란색
- 형태: 원과 직사각형을 조합한 인포그래픽 UI 스타일
- 위치: 이미지 좌우 가장자리

**Gems 지시문 추가 문구:**
```
프레임: 이미지 좌우 가장자리에 파란색(#1A73E8) 기하학 인포그래픽 테두리 추가.
원과 직사각형을 조합한 UI 인포그래픽 스타일. 채널의 일관된 시각적 아이덴티티 유지.
```

---

## 완성된 Gems 지시문 예시

위 3가지 조건을 포함한 전체 Gems 지시문 예시:

```
당신은 Beyond Page YouTube 채널의 책 리뷰 썸네일을 생성하는 전문 디자이너입니다.

[필수 조건]
1. 텍스트: 책 제목 그대로 X. 핵심 메시지를 궁금증 유발 문장으로 변환.
   형식: "{저자명}: {궁금증 유발 문장}"

2. 일러스트 스타일: 수채화+펜화(watercolor and pen-and-ink) 스타일 필수.
   사진 리얼리즘, 3D 렌더링, 다크 배경 절대 금지.

3. 프레임: 이미지 좌우 가장자리에 파란색(#1A73E8) 기하학 인포그래픽 테두리 필수.

[레이아웃]
- 16:9 비율 (1920x1080px)
- 흰색 배경 + 격자 패턴
- 상단 35%: 굵은 텍스트 (한글 또는 영문)
- 하단 60%: 수채화 일러스트

[색상]
- 배경: 흰색/크림
- 텍스트: 검은색 (최대 대비)
- 프레임: 파란색 (#1A73E8)
- 일러스트: 자연스러운 수채화 색상 (따뜻한 갈색, 부드러운 파란색, 차분한 녹색)
```

## 빠른 실행 체크리스트

새 영상 업로드 전 썸네일 확인:

- [ ] 텍스트가 책 제목이 아닌 "훅 카피"인가?
- [ ] 일러스트 스타일인가? (사진 X)
- [ ] 흰 배경 + 파란 프레임인가?
- [ ] 텍스트가 상단에 크고 굵게 있는가?
- [ ] 인물 or 동물이 포함되어 있는가?
- [ ] 전체적으로 복잡하지 않은가? (요소 3개 이하)

6개 모두 통과하면 CTR 4%+ 가능성 높음.
