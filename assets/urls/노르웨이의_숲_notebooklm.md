## 📝 사용 방법

1. 위 URL 블록을 전체 선택 (Cmd+A / Ctrl+A)
2. 복사 (Cmd+C / Ctrl+C)
3. NotebookLM에서 '소스 추가' > 'URL' 선택
4. 붙여넣기 (각 URL이 자동으로 인식됩니다)
