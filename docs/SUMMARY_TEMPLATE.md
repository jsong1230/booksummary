# 유튜브 롱폼 영상용 책 요약 템플릿

## 개요

이 템플릿은 유튜브 롱폼 영상용 5분 분량의 오프닝 서머리를 생성하기 위한 가이드입니다.

## 구조

```
[HOOK]
(짧고 강렬한 문장)

[SUMMARY]
(5분 분량 핵심 요약)

[BRIDGE]
(다음 NotebookLM 분석으로 자연스럽게 넘어가는 연결 문장)
```

## 요구사항

### 1. 강력한 Hook 문장으로 시작

시청자가 3초 만에 빠져들게 하는 문장이어야 합니다.

**Hook 스타일 옵션:**
- **질문형**: "만약 당신의 인생을 바꾼 문장이 단 한 줄이라면, 이 책은 그 한 줄을 제공합니다."
- **충격적인 통계**: "이 책이 전 세계에서 2000만 권 팔린 이유는, 우리가 모두 같은 고민을 하기 때문입니다."
- **역설**: "상상해보세요. 단 5초의 선택이 앞으로 5년을 결정한다면?"
- **감정적 장면**: "이 이야기는 시작부터 비극입니다. 그런데 모두가 공감합니다."

**Hook 작성 가이드:**
- 첫 3초 안에 시청자의 호기심을 자극해야 함
- 책의 가장 흥미롭고 충격적인 요소 활용
- 미스터리, 반전, 중요한 장면, 논쟁적 질문 등 사용
- 10-20초 분량의 강렬한 오프닝

### 2. 전체 내용을 한눈에 보여주는 5분 서머리

**핵심 구성 요소:**
- **주제/핵심 메시지**: 1~2문장으로 정리
- **주요 아이디어**: 3~5개를 뼈대만 남기고 요약
- **예시나 짧은 일화**: 1개 포함하면 좋음

**요약 작성 가이드:**
- 반드시 최소 750단어 이상 작성 (5분 분량 기준, 분당 150단어)
- 책의 주요 내용과 줄거리를 상세히 포함
- 주요 등장인물과 배경을 자세히 소개
- 책의 핵심 주제와 메시지를 깊이 있게 설명
- 중요한 사건이나 전개를 충분히 요약
- 자연스럽게 읽을 수 있는 문체로 작성

### 3. NotebookLM 상세 요약과 자연스럽게 연결되도록 마무리

**Bridge 문장 예시:**
- "지금부터 더 깊은 내용을 살펴보겠습니다."
- "이제 이 책의 숨겨진 의미와 작가의 의도를 자세히 분석해보겠습니다."
- "앞으로 이 작품이 우리에게 던지는 질문들을 하나씩 살펴보겠습니다."

**Bridge 작성 가이드:**
- 다음 NotebookLM 분석으로 자연스럽게 넘어가도록 연결
- 시청자가 계속 시청하도록 유도
- 간결하고 명확한 전환 문장

### 4. 톤과 문체

- **간결하고 직설적**: 불필요한 수식어 제거
- **영상 설명에 적합**: 듣기 쉬운 문체
- **존댓말 사용** (한국어의 경우)
- **격식 있는 문체**: 반말이나 구어체 사용 금지

## 사용 예시

### 입력 데이터
```
{{요약소스}}
(NotebookLM 요약 or 웹 기사 모음 or 텍스트)
```

### 출력 포맷
```
[HOOK]
만약 당신의 인생을 바꾼 문장이 단 한 줄이라면, 이 책은 그 한 줄을 제공합니다.

[SUMMARY]
(5분 분량의 상세한 요약 내용...)

[BRIDGE]
지금부터 더 깊은 내용을 살펴보겠습니다.
```

## Hook 스타일 참고 예시

1. **질문형**
   - "만약 당신의 인생을 바꾼 문장이 단 한 줄이라면, 이 책은 그 한 줄을 제공합니다."
   - "상상해보세요. 단 5초의 선택이 앞으로 5년을 결정한다면?"

2. **충격적인 통계**
   - "이 책이 전 세계에서 2000만 권 팔린 이유는, 우리가 모두 같은 고민을 하기 때문입니다."

3. **역설**
   - "이 이야기는 시작부터 비극입니다. 그런데 모두가 공감합니다."

4. **감정적 장면**
   - "1980년 광주에서 일어난 충격적인 사건은..."
   - "이 책의 주인공은 마지막 장에서..."

## 코드에서 사용하기

이 템플릿은 `src/08_generate_summary.py`의 `generate_summary()` 함수에서 사용됩니다.

프롬프트에 이 템플릿의 요구사항이 반영되어 있으며, `use_engaging_opening=True` 옵션을 사용하면 Hook 스타일의 오프닝이 생성됩니다.
