# YouTube 영상 검색 최적화 가이드

## 개요
이 문서는 YouTube 영상의 해시태그, 제목, 메타데이터를 검색에 최적화하는 방법을 설명합니다.

## 주요 개선 사항

### 1. 해시태그 최적화

#### 개선된 기능
- **장르별 특화 태그 자동 감지**: 책의 장르(소설, 논픽션, 철학, 과학 등)를 자동으로 감지하여 관련 태그 추가
- **검색량 높은 키워드 우선 배치**: 검색량이 높은 기본 태그를 앞쪽에 배치
- **트렌딩 키워드 추가**: 최신 검색 트렌드에 맞는 태그 자동 추가
- **기관 태그 최적화**: 뉴욕타임즈, 아마존, 하버드 등 검색량 높은 기관 태그 포함

#### 태그 우선순위
1. **기본 태그** (검색량 높은 순서)
   - 한글: 책리뷰, 독서, 북튜버, 책추천, 독서법, 책읽기 등
   - 영문: BookReview, Reading, BookTube, BookRecommendation 등

2. **장르 태그** (책의 장르에 따라 자동 추가)
   - 소설: 소설, 문학, 소설리뷰, 문학작품 등
   - 논픽션: 논픽션, 에세이, 수필, 인문학 등
   - 철학: 철학, 철학서, 철학책 등
   - 과학: 과학, 과학서, 과학책 등

3. **기관 태그** (검색 최적화)
   - 미디어: 뉴욕타임즈, 아마존, 타임지 등
   - 대학: 하버드, 시카고대학교, 서울대학교 등
   - 문학상: 노벨문학상, 퓰리처상, 맨부커상 등

4. **책 제목/작가 태그**
   - 책 제목, 작가 이름 기반 태그

5. **트렌딩 태그**
   - 책추천2024, 독서챌린지, 독서모임 등

#### 태그 제한
- YouTube 태그 최대 길이: 500자
- 태그 개수: 약 30-40개 (태그당 평균 10-15자)
- 각 태그 최대 길이: 30자

### 2. 제목 최적화

#### 개선된 기능
- **작가 이름 자동 포함**: 제목에 작가 이름을 포함하여 검색 범위 확대
- **검색 키워드 앞쪽 배치**: 검색량 높은 키워드를 제목 앞부분에 배치
- **대체 제목 포함**: 책의 대체 제목을 괄호 안에 포함하여 검색 가능성 증가

#### 제목 형식
- **한글 영상**: `[한국어] {책제목} 책 리뷰 {작가명} | [Korean] {영어제목} Book Review`
- **영문 영상**: `[English] {영어제목} Book Review {작가명} | [영어] {한글제목} 책 리뷰`

#### SEO 최적화 팁
1. **핵심 키워드 포함**: "책 리뷰", "Book Review" 등 검색량 높은 키워드 필수 포함
2. **작가 이름 포함**: 작가 이름을 포함하면 작가 검색 시에도 노출 가능
3. **대체 제목 활용**: 책의 다른 이름이나 부제목을 포함하여 검색 범위 확대
4. **언어 표시**: [한국어], [Korean] 등 언어 표시로 다국어 검색 대응

### 3. 메타데이터(설명) 최적화

#### 개선된 기능
- **키워드 자연스럽게 포함**: 설명 첫 부분에 검색 키워드 자연스럽게 배치
- **구조화된 정보**: 타임스탬프, 챕터 마커 등으로 사용자 경험 향상
- **관련 키워드 밀도 최적화**: 설명 전체에 관련 키워드를 자연스럽게 배치

#### 설명 구조
1. **YouTube 챕터 마커** (맨 앞)
   ```
   0:00 요약 (Summary)
   1:42 NotebookLM 상세 분석
   ```

2. **검색 키워드 포함 헤더**
   ```
   📚 책 리뷰 영상 | 독서 | 북튜버 | 책추천
   ```

3. **영상 구성 설명**
   - GPT 요약, NotebookLM 분석 등 영상 내용 설명

4. **책 소개**
   - 책의 주요 내용, 작가 배경, 작품 의미 등
   - 검색 키워드를 자연스럽게 포함

5. **작가 정보**
   - 작가 이름, 출간일 등

6. **상호작용 유도**
   - 좋아요, 구독, 댓글 유도 문구

7. **해시태그**
   - 설명 끝부분에 해시태그 포함

#### SEO 최적화 팁
1. **첫 125자 중요**: YouTube 검색 결과에서 보이는 부분이므로 핵심 키워드 포함
2. **키워드 밀도**: 설명 전체에 관련 키워드를 자연스럽게 2-3회 포함
3. **구조화된 정보**: 타임스탬프, 챕터 마커 등으로 사용자 경험 향상
4. **관련 키워드**: 책 제목, 작가 이름, 장르 등 관련 키워드 자연스럽게 포함

## 자동 생성 기능

현재 시스템은 다음을 자동으로 수행합니다:

1. **장르 자동 감지**: `book_info`의 `categories`와 `description`을 분석하여 장르 감지
2. **장르별 태그 자동 추가**: 감지된 장르에 맞는 태그 자동 추가
3. **작가 이름 자동 포함**: 제목과 설명에 작가 이름 자동 포함
4. **검색 키워드 자동 배치**: 검색량 높은 키워드를 앞쪽에 배치
5. **트렌딩 태그 자동 추가**: 최신 검색 트렌드에 맞는 태그 추가

## 수동 최적화 팁

### 1. 제목 개선
- 책의 핵심 키워드를 앞쪽에 배치
- 검색량 높은 키워드 우선 포함
- 제목이 너무 길면 핵심 키워드만 남기기

### 2. 설명 개선
- 첫 문장에 핵심 키워드 포함
- 설명 전체에 관련 키워드 자연스럽게 배치
- 사용자가 검색할 만한 키워드 포함

### 3. 태그 개선
- 검색량 높은 태그 우선 포함
- 장르별 특화 태그 추가
- 트렌딩 키워드 활용
