## 🎯 기본 프롬프트

### 1. 책 요약 생성
```
이 책의 주요 내용을 3-5분 분량의 오디오 대본으로 요약해주세요. 
일당백 팟캐스트 스타일로 작성해주세요. 
친근하고 대화하듯이 자연스러운 톤으로 작성하고, 
핵심 내용을 쉽게 이해할 수 있도록 설명해주세요.
```

### 2. 핵심 메시지 추출
```
이 책의 핵심 메시지와 주요 인사이트를 5가지로 정리해주세요. 
각 메시지에 대해 간단한 설명을 추가해주세요.
```

### 3. 작가 스타일 분석
```
무라카미 하루키의 작품 스타일과 특징을 분석해주세요. 
이 책에서 나타나는 그의 문학적 기법과 주제 의식을 설명해주세요.
```

## 🎬 영상 제작용 프롬프트

### 4. 오디오 대본 생성 (10-15분)
```
이 책에 대한 10-15분 분량의 오디오 대본을 작성해주세요.

구조:
1. 인트로 (30초): 책 소개와 화제성
2. 작가 소개 (1분): 무라카미 하루키에 대한 간단한 소개
3. 책의 배경과 설정 (2분)
4. 주요 등장인물 소개 (2분)
5. 핵심 내용과 주제 (5-7분)
6. 마무리 (1분): 독자에게 전하는 메시지

스타일:
- 일당백 팟캐스트처럼 친근하고 대화하듯이
- 전문 용어는 쉽게 풀어서 설명
- 구체적인 예시와 비유 사용
- 자연스러운 문장 흐름
- 감정을 자극하는 표현 사용
```

### 5. 챕터별 요약
```
이 책을 챕터별로 나누어 각 챕터의 핵심 내용을 1-2분 분량으로 요약해주세요.
각 챕터의 중요한 장면이나 대화를 인용하면서 설명해주세요.
```

### 6. 인물 분석
```
이 책의 주요 등장인물들을 분석해주세요. 
각 인물의 성격, 역할, 그리고 이야기에서의 의미를 설명해주세요.
```

## 🎙️ 팟캐스트 스타일 프롬프트

### 10. 대화형 리뷰 스크립트
```
이 책에 대한 대화형 리뷰 스크립트를 작성해주세요.

형식:
- 호스트가 질문하고, AI가 답변하는 형식
- 자연스러운 대화 흐름
- "그렇다면", "흥미롭네요", "더 자세히 설명해주세요" 같은 자연스러운 연결어 사용
- 15-20분 분량

내용:
- 책의 첫인상
- 읽으면서 느낀 점
- 인상 깊었던 장면
- 작가의 메시지
- 추천 대상
```

### 11. 질문과 답변 형식
```
이 책에 대해 청취자들이 궁금해할 만한 질문 10가지를 만들고, 
각 질문에 대해 상세하게 답변해주세요.

예시 질문:
- 이 책의 제목 '노르웨이의 숲'은 무슨 의미인가요?
- 주인공 와타나베의 성격은 어떤가요?
- 이 작품의 주제는 무엇인가요?
- 왜 이 책이 많은 사람들에게 사랑받는가요?
```

## 🎯 최종 오디오 대본 생성 프롬프트 (권장)

```
다음 요구사항에 맞춰 '노르웨이의 숲'에 대한 오디오 대본을 작성해주세요:

**길이**: 12-15분 분량
**스타일**: 일당백 팟캐스트 스타일
**톤**: 친근하고 대화하듯이, 전문적이지만 접근하기 쉽게

**구성**:
1. 인트로 (30초)
   - 책 제목과 작가 소개
   - 오늘 다룰 내용 미리보기

2. 작가 소개 (1분)
   - 무라카미 하루키는 누구인가
   - 그의 대표작과 문학적 특징

3. 책의 배경 (1분)
   - 작품이 쓰여진 시대적 배경
   - 작품의 설정과 분위기

4. 주요 등장인물 (2분)
   - 주인공 와타나베
   - 나오코, 미도리 등 주요 인물
   - 각 인물의 특징과 역할

5. 핵심 내용 (6-8분)
   - 이야기의 주요 흐름
   - 중요한 장면과 대화
   - 작가가 전달하고자 하는 메시지
   - 작품의 주제와 의미

6. 문학적 특징 (2분)
   - 무라카미 하루키의 독특한 문체
   - 상징과 은유
   - 서사 구조의 특징

7. 마무리 (1분)
   - 이 책을 읽어야 하는 이유
   - 추천 대상
   - 마무리 멘트

**작성 요청사항**:
- 자연스러운 구어체 사용
- 전문 용어는 쉬운 말로 풀어서 설명
- 구체적인 예시와 비유 활용
- 감정을 자극하는 표현 사용
- 청취자가 몰입할 수 있는 흥미로운 구성
- 각 섹션 간 자연스러운 전환
- 핵심 내용을 놓치지 않으면서도 지루하지 않게
```

## 💡 프롬프트 사용 팁

1. **단계별 접근**: 먼저 기본 요약을 받고, 그 다음 상세 분석을 요청하세요.
2. **반복 수정**: 생성된 내용을 바탕으로 "더 자세히", "더 쉽게" 등으로 수정 요청하세요.
3. **스타일 조정**: "더 친근하게", "더 전문적으로" 등으로 톤을 조정하세요.
4. **길이 조절**: "5분 분량으로", "10분 분량으로" 등으로 길이를 지정하세요.
5. **구체적 요청**: "예시를 들어서", "비유를 사용해서" 등 구체적으로 요청하세요.
